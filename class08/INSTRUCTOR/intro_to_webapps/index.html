Ports

Computers have thousands of network Ports. These ports send and recieve messages over a network.

TCP/IP

TCP/IP

Transmission Control Protocol builds upon basic Internet Protocol. These protocols allow computers to establish efficient port-to-port connections.

HTTP

Once their host computers are communicating via TCP/IP, Client and Server web applications communicate using HyperText Transfer Protocol. This protocol instructs how the apps interact.

HTTP

Using HyperText Transfer Protocol, a web client may request action from a server application, and expect a response back.
